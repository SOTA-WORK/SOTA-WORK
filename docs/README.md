# SOTA

## CV

最先进的深度学习库可以产生计算机视觉。它旨在收集和维护有关计算机视觉最新发展的最新信息，促进深度学习的研究工作。

与协作跟踪研究进展的其他尝试不同，此存储库提供定量评估的汇总结果。这种做法可以大大简化初始文献检索和准备对自己结果的比较研究。
